    Python Condition

    python support usual logical condition

  • Equal:a==b
  • Not Equal:a!=b

    • Elif keyword is "if the previous conditions were not true, then try this condition"

    # input a=5 b=5 if a > b: print("b is greater than a") elif a==b: print("a and b are equal") #output a and b are equal

    Else "else keyword catches anything which isn't caught by the preceding conditions"

    #input a=5 b=25 if b > a: print(b is greater than a ); elif a==b: print(a is equal to b); else: print(a is greater than b) #output a is greater than b
    Python Loops
  • while loop
  • for loop

    • while loop we can execute a set of statements as long as a condition is true

    #input i=1 while i < 6: print(i) i+=1 #output 1 2 3 4 5

    break statement we can stop the loop even if the while condition is true

    #input i=1 while i < 6: print(i) if(i==3) break i+=1 #output 1 2 3

    A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string)

  • This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages. With the for loop we can execute a set of statements, once for each item in a list
    • tuple, set etc

    Print each name in the list: names=["saran","mathew","modi"] for i in names: print(i) #output saran mathew modi
    Python Function

    A function is a block of code which only runs when it is called.You can pass data, known as parameters, into a function can return data as a result

  • create a function
    • created using def keyword def my_function(): print("Hello from a function") my_function() #calling a function #output Hello from a function
  • Passing a list as a argument

    • You can send any data types of argument to a function (string, number, list, dictionary etc.), and it will be treated as the same data type inside the function. E.g. if you send a List as an argument, it will still be a List when it reaches the function

    def my_function(food): for i in food: print(i) fruits=["apple","orange","mango"] my_function(fruits) #output apple orange mango
  • range()function using for loop

    • To loop through a set of code a specified number of times, we can use the range() function, The range() function returns a sequence of numbers, starting from 0 by default, and increments by 1 (by default), and ends at a specified number.

    for i in range(6): print(i) # output 0 1 2 3 4 5
